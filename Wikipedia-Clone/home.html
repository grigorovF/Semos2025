Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and JavaScript.

History

HTML was created by Tim Berners-Lee in 1991. The first official specification, HTML 2.0, was published in 1995. Subsequent versions improved functionality and compatibility.

Markup

HTML documents are built from elements marked up with tags such as <p>, <a>, and <img>. These define structure and meaning of the content.

Examples

<!doctype html>
<html>
  <head><title>Example</title></head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
